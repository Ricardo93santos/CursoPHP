Obter dados de Formulários no PHP

Para tratar dados de formulário em PHP devemos usar as tags HTML de formulário, onde os usuários inserirão os dados solicitados. Os formulários usam elementos como "‹form›", "‹input›", "‹select›", "‹textarea›" e entre outros. Para saber mais sobre os elementos de formulário acesse o site do W3Scholls pelo link HTML forms.

Na tag "‹form›" escpecificamos os método de envio usando o atributo "method" e o atributo "action" para espesificar para onde os dados do formulário devem ser enviados quando o formulário for submetido. O método mais comim é o "POST", que envia os dados do formulário de forma mais segura e permite enviar uma quantidade maior de dados. O metodo "GET" tmabém pode ser usado, mas ele exibe os dados na URL, o que pode ser menos seguro.

    EX:
    ‹form method = "post" action = "processar.php"›
      ‹laber for = "nome"›Nome:‹/label›
      ‹input type = "text" name = "nome" id = "nome"›
      ‹laber for = "sobrenome"›Sobrenome:‹/label›
      ‹input type = "text" name = "sobrenome" id = "sobrenome"›
      ‹input type = "submit" value = "Enviar"›
    ‹/form›

No PHP os dados enviados são armazenados nos arrays superglobal "&_POST" (quando o metodo é POST)ou no "&_GET" (quando o metodo é GET), depois são tratados conforme a necessidade dos sistema e a lóagica do script.

    EX: PHP (processar.php)
    ‹?php
    if($_SERVER['REQUEST_METHOD']=="POST"){
      $nome = $_POST["nome"];
      $sobrenome = $_POST["sobrenome"];
      if($nome == "" || $sobrenome == ""){
        echo "Informe o nome e seu sobrenome";
      }else{
        echo "Olá, ‹strong›$nome $sobrenome‹/strong›, seu formulário foi enviado com sucesso!";
      }
    }
