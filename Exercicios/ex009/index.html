Superglobais PHP

Superglobais no PHP são arrays especiais que armazenam informações globais do ambiente do servidor e da solicitação HTTP, exitem 8 superglobais no PHP. Elas são automaticamente acessíveis de qualquer parte do script PHP, sem a necessidade de declaração adicional.

As superglobais são:

  1. $_GET: Usada para coletar enviados para o script via paraametros de URL, é frequentimente usada para acessar dados de consulta em URLs.
    Ex: $_GET['nome']
  2. $_POST: Usada para coletar dados enviados por um formulário HTML, semelhante ao metodo GET, esses dados são invisíveis na URL, o que é útil para informações sensíveis.
    Ex: $_POST['senha']
  3. $_REQUEST: Combina dados das superglobais '$_GET', '$_POST' e '$_COOKIE'. Porém seu uso não é recomendado, pois pode ser menos seguro e mais difícil de rastrear a origem dos dados.
    Ex: $_REQUEST['senha']
  4. $_SESSION: Usada para manter informações da sessão do usuário em várias páginas. É útil para armazenar dados como informações de login, carrinhos de compras, etc.
    Ex: session_start();//inicia uma sessão $_SESSION['user_id'] = 157;
  5. $_COOKIE: Contem todos os cookies enviados pelo navegador do usuário. São pequenos dados armazenados localmente e são frequentemente usados para lembrar preferências ou informações de login.
    Ex: Se houver um cookie chamado "preferencia" definido no navegador, pode ser acessado usando $_COOKIE['preferencia'];
  6. $_FILES: Usada para coletar informações sobre arquivos enviados por formulários. Ela fornece detalhes sobre o arquivo, como nome, tipo e local temporário no servidor.
    Ex: enctype = "multipart/form-data"
  7. $_SERVER: Fornece informações sobre o servidor web, a solicitação HTTP e o ambiente em que o seu script PHP está sendo executado.
    Ex: $_SERVER['REMOTE_ADDR']; //endereço de IP do usuário da página
  8. $_ENV: Contem variáveis de ambiente definidas no servidor. Elas podem ser úteis para acessar configurações específicas do ambiente em que o PHP está sendo executado.
    Ex: Se hover uma variável de ambiente chamada "DATABASE_PASSWORD", pode acessa-la usando $_ENV["DATABASE_PASSWORD"];
